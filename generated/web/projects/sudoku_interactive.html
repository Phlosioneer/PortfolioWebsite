Interactive Sudoku

A Sudoku game that teaches the player advanced techniques

Interactive Sudoku is an app that helps players improve their Sudoku skills. While there are many Sudoku apps out there, and many solvers as well, this app is different. The app notices the techniques that the player is using to solve puzzles, and how often the player is able to notice those moves, and saves those as a profile of the player. The app then uses that profile to provide targeted hints, and to generate puzzles that teach the player new skills.

This project is a work in progress, and there are technical challenges that I haven't resolved yet. I have basic Sudoku entering and playing functionality, and I have a list of techniques to implement as part of the skill recognition system. The rest has not been implemented yet.
